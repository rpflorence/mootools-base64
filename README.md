Base64
======

![icon](http://github.com/rpflorence/Base64/icon.png)

String methods for encoding and decoding Base64 data.

How to use
----------

    var str = "General Conference";
    var encoded = str.toBase64();
    var decoded = encoded.decodeBase64();
